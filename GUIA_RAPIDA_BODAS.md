📥 EXPORTACIÓN A CSV - MODO DE USO

┌─ OPCIÓN 1: Desde Pantalla Principal ─────────────────────────────────────┐
│                                                                             │
│ 1. Abre la app y entra a la sección de Bodas 💍                           │
│ 2. Toca el menú superior (⋮) en la esquina superior derecha              │
│ 3. Selecciona:                                                             │
│    - "📥 Exportar Checklist (CSV)"      → descarga checklist.csv         │
│    - "📥 Exportar Presupuesto (CSV)"    → descarga presupuesto.csv       │
│    - "📥 Exportar Proveedores (CSV)"    → descarga proveedores.csv       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ OPCIÓN 2: Desde Pantalla de Proveedores ────────────────────────────────┐
│                                                                             │
│ 1. Ve a Wedding → Proveedores 👨‍💼                                          │
│ 2. Toca el ícono de descarga (📥) en la esquina superior derecha         │
│ 3. El CSV se genera automáticamente                                        │
│ 4. Opción de verlo en pantalla o descargar                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

ESTRUCTURA DE ARCHIVOS CSV:

📄 CHECKLIST.CSV
├─ Tarea: Nombre de la tarea
├─ Categoría: Tipo (Venue, Catering, Fotos, etc)
├─ Completada: Sí/No
├─ Fecha Límite: dd/mm/yyyy
├─ Prioridad: Alta/Media/Baja
└─ Notas: Información adicional

📄 PRESUPUESTO.CSV
├─ Concepto: Nombre del gasto
├─ Categoría: Tipo de gasto
├─ Estimado: Cantidad planeada
├─ Pagado: Cantidad pagada
├─ Pendiente: Lo que falta
├─ % Pagado: Porcentaje completado
├─ Proveedor: Nombre del proveedor
├─ Fecha Pago: dd/mm/yyyy
├─ Método: Efectivo/Tarjeta/Transferencia
└─ Notas: Información adicional

📄 PROVEEDORES.CSV
├─ Servicio: Tipo de servicio
├─ Proveedor: Nombre
├─ Teléfono: +34 xxx xxx xxx
├─ Email: correo@example.com
├─ Precio: $
├─ Rating: x/5
├─ Contratado: Sí/No
└─ Fecha Contratación: dd/mm/yyyy

📊 CÓMO USAR EN EXCEL

PASO 1: DESCARGAR
└─ Descarga el archivo CSV desde la app

PASO 2: ABRIR EN EXCEL
├─ Opción A (Más fácil):
│  └─ Haz doble clic en el archivo CSV
└─ Opción B (Más control):
   ├─ Abre Excel
   ├─ Archivo → Abrir
   ├─ Selecciona el archivo CSV
   └─ Excel auto-detecta el formato

PASO 3: IMPORTAR DATOS
├─ Excel reconoce automáticamente:
│  ├─ Las columnas (encabezados)
│  ├─ Los tipos de datos
│  └─ El delimitador (comas)

PASO 4: PERSONALIZAR EN EXCEL
├─ Puedes:
│  ├─ Cambiar colores y formato
│  ├─ Agregar fórmulas (sumas, promedios)
│  ├─ Crear gráficos
│  ├─ Filtrar y ordenar
│  ├─ Agregar más columnas
│  └─ Guardar como .xlsx (Excel nativo)

PASO 5: SINCRONIZACIÓN (OPCIONAL)
├─ Edita el CSV en Excel
├─ Guarda como CSV de nuevo
├─ Copia el contenido actualizado
└─ Importa nuevamente en la app (próximo: implementar)

💡 TIPS Y TRUCOS

🎯 ORGANIZACION EFICIENTE
├─ Usa el Checklist con fechas límite para no olvidar nada
├─ Asigna prioridades (Alta/Media/Baja) a tareas
├─ Exporta regularmente a CSV para tener respaldo
├─ Usa el Presupuesto para evitar gastar más de lo planeado
└─ Recolecta todos los emails de proveedores en Proveedores

🎵 PARA EL DÍA DE LA BODA
├─ Imprime el Itinerario y comparte con el coordinador
├─ Ten a mano los números de emergencia de Proveedores
├─ Lleva el checklist del día anterior impreso
├─ Confirma los horarios con 24 horas de anticipación
└─ Delega responsabilidades usando las notas

📸 PARA DESPUÉS DE LA BODA
├─ Carga todas las fotos al Álbum de Fotos
├─ Marca las fotos destacadas (⭐)
├─ Agradece a todos los proveedores (usa contactos)
├─ Exporta fotos a compartir con invitados
└─ Guarda un CSV final con todo como registro

❓ PREGUNTAS FRECUENTES

P: ¿Dónde guardo los CSV descargados?
R: Depende de tu dispositivo:
   - Android: Carpeta Downloads
   - iPhone: Archivos → Descargas
   - Web: Carpeta de Descargas del navegador

P: ¿Puedo sincronizar cambios entre app y Excel?
R: Próximamente. Por ahora exporta, edita y vuelve a exportar para actualizar.

P: ¿Los datos se guardan automáticamente?
R: Sí, se guardan en la app. Exporta regularmente a CSV para respaldo.

P: ¿Puedo compartir el CSV con mi pareja?
R: Sí. Descarga el archivo y comparte por:
   - Email
   - WhatsApp
   - Google Drive
   - Dropbox
   - O copia el contenido directamente

P: ¿Qué pasa si pierdo el teléfono?
R: Los datos se guardarán en la nube (próximamente). Por ahora:
   - Haz backup regular a CSV
   - Guarda los CSV en Google Drive/Dropbox

P: ¿Cómo borro un proveedor/regalo/plato?
R: Toca el ícono de opciones (⋮) o desliza para eliminar.
   (Próxima: confirmación antes de eliminar)

P: ¿Puedo tener múltiples bodas?
R: Actualmente solo soporta una boda. Próximamente: múltiples perfiles.

🎓 TUTORIAL COMPLETO (5 MIN)

1. CONFIGURACIÓN INICIAL (DÍA 1)
   ├─ Abre Wedding → Invitación
   ├─ Coloca fecha, hora y lugar
   ├─ Ve a Wedding → Checklist
   ├─ Comienza a agregar tareas por categoría
   └─ Asigna fechas límite y prioridades

2. INVITADOS Y REGALOS (SEMANA 1)
   ├─ Entra a Wedding → Invitados
   ├─ Agrega lista de invitados
   ├─ Ve a Wedding → Mesa de Regalos
   ├─ Agrega artículos deseados
   └─ Comparte registry con familia

3. SERVICIOS Y PRESUPUESTO (SEMANA 2-4)
   ├─ Wedding → Proveedores
   ├─ Agrega fotógrafo, catering, DJ, etc
   ├─ Ingresa contactos y presupuestos
   ├─ Wedding → Presupuesto
   ├─ Crea gastos por categoría
   └─ Exporata a CSV para seguimiento

4. DETALLES (SEMANA 5-8)
   ├─ Wedding → Menú (diseña con catering)
   ├─ Wedding → Flores (coordina decoración)
   ├─ Wedding → Look (vestido, traje, estilismo)
   ├─ Wedding → Playlist (crea lista con DJ)
   └─ Wedding → Hospedaje (comparte con invitados)

5. SEMANA PREVIA
   ├─ Wedding → Itinerario (revisa cronograma)
   ├─ Wedding → Checklist (marca completadas)
   ├─ Exporta Checklist para el día (imprime)
   ├─ Reconfirma todos los proveedores
   └─ Guarda CSV de respaldo

6. DESPUÉS DE LA BODA
   ├─ Wedding → Álbum (sube fotos)
   ├─ Marca fotos destacadas (⭐)
   ├─ Agradece a proveedores (usa contactos)
   ├─ Exporta final de gastos
   └─ Comparte álbum con invitados (próximo)
